# C04 — Q-Table Dispatch Policy

## Purpose
Q-Table Dispatch Policy in the [[C-Series]]. Layer: **RL/Q-Learning**.

## Formula

### LaTeX
$$Q(s,a) \leftarrow Q(s,a) + \alpha_Q [r + \gamma_Q \max_{a'} Q(s',a') - Q(s,a)]$$

### Description
Q-learning update rule. State s = (SoC_bin, price_bin, occ_state); action a = (charge/discharge/idle). Learns optimal policy for simple asset control.

### Method
Tabular Q-learning, discrete state-action space
