# C02 — MPC (Model Predictive Control) Controller

## Purpose
MPC (Model Predictive Control) Controller in the [[C-Series]]. Layer: **RL/Q-Learning**.

## Formula

### LaTeX
$$\min_{u_{k:k+N}} J = \sum_{i=k}^{k+N} \ell(x_i, u_i) + V_f(x_{k+N})$$

### Description
MPC minimises cost J over rolling horizon N timesteps. State x includes T_in and SoC; control u includes P_HP, P_bat. Re-solves every Δt with latest measurements.

### Method
Rolling horizon MPC, receding window
