# C05 — DQN (Deep Q-Network) Controller

## Purpose
DQN (Deep Q-Network) Controller in the [[C-Series]]. Layer: **RL/Q-Learning**.

## Formula

### LaTeX
$$Q_\theta(s,a) \approx r + \gamma \max_{a'} Q_{\theta^-}(s',a')$$

### Description
DQN approximates Q-function with neural network θ. Target network θ⁻ provides stable training targets. Experience replay buffer size = 10,000 transitions.

### Method
Deep Q-Network, experience replay
