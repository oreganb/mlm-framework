# C07 — Multi-Agent RL Community Coordinator

## Purpose
Multi-Agent RL Community Coordinator in the [[C-Series]]. Layer: **RL/Q-Learning**.

## Formula

### LaTeX
$$J(\pi_1, ..., \pi_N) = E\left[\sum_{i=1}^N R_i(s, a_1,...,a_N)\right]$$

### Description
MARL coordination across N building agents. Each agent optimises its own policy π_i while considering shared grid constraints. Centralised training, decentralised execution (CTDE).

### Method
Multi-Agent Proximal Policy Optimisation (MAPPO)
