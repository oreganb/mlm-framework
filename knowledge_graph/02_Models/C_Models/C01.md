# C01 ‚Äî MILP Flexibility Engine

## Purpose
The core optimisation model. Computes upward and downward flexibility (Flex‚Üë/‚Üì) at each timestep across a rolling horizon H, subject to all physical, comfort, and market constraints. Produces the dispatch schedule for all controllable assets.

## Layer
**RL/Q-Learning** ‚Äî the deterministic optimiser at the heart of the decision layer. Receives determinized inputs from C03 (Determinization Bridge) and outputs control signals to all assets.

---

## Formula

### LaTeX ‚Äî Objective Function
$$\min_{\mathbf{x}_t} \sum_{k \in H} \left[ \alpha \cdot C_{energy}(P^{grid}_k, \pi_k) + \beta \cdot C_{comfort}(T^{in}_k) + \gamma \cdot C_{carbon}(P^{grid}_k, EF_k) \right]$$

**Net power at PCC:**
$$P^{grid}_k = P^{base}_k - P^{PV}_k - P^{bat,dis}_k + P^{bat,chg}_k - P^{shed}_k - P^{shift}_k + P^{HP}_k + P^{EV}_k$$

**Flexibility definitions:**
$$\text{Flex}^{\uparrow}_k = \max \Delta^{\uparrow}_k \quad \text{s.t. } P^{grid}_k - \Delta^{\uparrow}_k \geq P^{grid,min}_k$$
$$\text{Flex}^{\downarrow}_k = \max \Delta^{\downarrow}_k \quad \text{s.t. } P^{grid}_k + \Delta^{\downarrow}_k \leq P^{grid,max}_k$$

**Battery dynamics:**
$$SoC_{k+1} = SoC_k + \frac{\eta_{ch} P^{bat,chg}_k \Delta t}{E_{nom}} - \frac{P^{bat,dis}_k \Delta t}{\eta_{dis} E_{nom}}$$

**Thermal comfort (RC model coupling):**
$$T^{in}_{k+1} = a T^{in}_k + b_1 Q^{HVAC}_k + b_2 T^{out}_k + b_3 G_{solar,k}$$

### Python Pseudocode
```python
def milp_flexibility_engine(horizon_steps, dt_s,
                             weather_det, occupancy_det, price_det,
                             building_params, battery_params, pv_params,
                             weights_alpha, weights_beta, weights_gamma):
    """
    C01 MILP Flexibility Engine

    Inputs (all determinized ‚Äî from C03 Determinization Bridge):
      horizon_steps  : int, number of timesteps in horizon H (e.g., 96 for 24h at 15min)
      dt_s           : float, timestep in seconds (900 for 15min)
      weather_det    : dict, {T_out[k], G_POA[k]} ‚Äî determinized from W-Series (p10 or p50)
      occupancy_det  : dict, {occ_state[k], Q_occ[k]} ‚Äî from O-Series
      price_det      : dict, {price_DA[k], EF[k]} ‚Äî from M-Series
      building_params: dict from DS-BUILDING: R_env, C_zone, T_min, T_max
      battery_params : dict from DS-BATTERY: E_nom, eta_ch, eta_dis, SoC_min, SoC_max
      pv_params      : dict: P_PV[k] per timestep (from R01)
      weights_alpha  : float, energy cost weight  [w_alpha, default=0.5]
      weights_beta   : float, comfort weight       [w_beta, default=0.3]
      weights_gamma  : float, carbon weight        [w_gamma, default=0.2]
                       NOTE: alpha + beta + gamma must = 1.0

    Decision variables (per timestep k):
      P_HP[k]       : Heat pump power (W)
      P_EV[k]       : EV charging power (W)
      P_shed[k]     : Load shed (W)
      P_shift[k]    : Load shift (W)
      P_bat_chg[k]  : Battery charge (W)
      P_bat_dis[k]  : Battery discharge (W)
      SoC[k]        : Battery SoC (fraction)
      T_in[k]       : Indoor temperature (¬∞C)
      delta_up[k]   : Upward flexibility (W)
      delta_down[k] : Downward flexibility (W)
    """
    # Build LP/MILP problem
    prob = create_optimisation_problem()

    for k in range(horizon_steps):
        # Net power at PCC
        P_base = occupancy_det['Q_occ'][k] / COP_base  # [w=1/COP_base]
        P_PV = pv_params['P_PV'][k]                   # from R01

        P_grid = (P_base - P_PV
                  - P_bat_dis[k] + P_bat_chg[k]
                  - P_shed[k] - P_shift[k]
                  + P_HP[k] + P_EV[k])

        # Objective terms
        C_energy = price_det['price_DA'][k] * P_grid * (dt_s/3600)
        C_comfort = penalty_comfort(T_in[k], T_min, T_max)  # [w_beta = 0.3]
        C_carbon = price_det['EF'][k] * P_grid * (dt_s/3600)

        prob.minimize(weights_alpha * C_energy      # [w_alpha = 0.5]
                    + weights_beta  * C_comfort     # [w_beta  = 0.3]
                    + weights_gamma * C_carbon)     # [w_gamma = 0.2]

        # Battery dynamics constraint
        prob.add_constraint(SoC[k+1] == SoC[k]
            + eta_ch * P_bat_chg[k] * (dt_s/3600) / E_nom   # [w=eta_ch/E_nom]
            - P_bat_dis[k] * (dt_s/3600) / (eta_dis * E_nom)) # [w=1/(eta_dis*E_nom)]

        # Thermal RC constraint (from T01 parameters)
        a = exp(-dt_s / (R_env * C_zone))                    # [w = RC time constant]
        b1 = R_env * (1 - a)                                  # [w = R_env*(1-a)]
        b2 = (1 - a)                                          # [w = (1-a)]
        b3 = R_env * (1 - a)                                  # [w = R_env*(1-a)]
        prob.add_constraint(T_in[k+1] == a*T_in[k]
            + b1 * Q_HVAC[k]
            + b2 * T_out[k]
            + b3 * G_solar[k])

        # Flexibility LP (solve twice: up and down)
        prob.add_constraint(delta_up[k] >= 0)
        prob.add_constraint(delta_down[k] >= 0)

    return prob.solve()
```
