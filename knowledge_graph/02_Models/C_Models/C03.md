# C03 — Determinization Bridge

## Purpose
Determinization Bridge in the [[C-Series]]. Layer: **RL/Q-Learning**.

## Formula

### LaTeX
$$x_{det}^{(q)} = F_X^{-1}(q), \quad q \in \{0.1, 0.5, 0.9\}$$

### Description
Converts stochastic forecast distributions from W/O/M-Series into deterministic inputs for C01 MILP. Three methods: quantile (fast), ensemble (robust), chance-constraint (risk-aware).

### Method
Quantile, Ensemble, Chance-Constraint
