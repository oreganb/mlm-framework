# C09 — Fragmented Islanding Controller

## Purpose
Fragmented Islanding Controller in the [[C-Series]]. Layer: **RL/Q-Learning**.

## Formula

### LaTeX
$$u_{island} = \arg\min_u |P_{gen,segment} - P_{load,segment}| \text{ s.t. } V \in [V_{min}, V_{max}]$$

### Description
Islanding controller balances local generation and load within each island segment identified by G08. Minimises frequency deviation and voltage deviation.

### Method
Rule-based + RL, segment-aware dispatch
