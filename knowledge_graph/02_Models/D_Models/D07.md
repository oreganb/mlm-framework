# D07 — Outlier Detection Model

## Purpose
Outlier Detection Model in the [[D-Series]]. Layer: **AI/ML**.

## Formula

### LaTeX
$$outlier = \mathbf{1}[|x - \mu| > 3\sigma] \lor \mathbf{1}[x \notin [Q_1 - 1.5 IQR, Q_3 + 1.5 IQR]]$$

### Description
Statistical outlier detection using Z-score (>3σ) and IQR methods in ensemble. Applied to all time-series inputs before model ingestion.

### Method
Z-score, IQR, Isolation Forest ensemble

## Weights & Parameters

- **W-D07-IQR:** IQR multiplier = 1.5 (standard Tukey fence)
