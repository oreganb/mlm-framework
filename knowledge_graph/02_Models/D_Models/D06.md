# D06 — Missing Data Imputation Model

## Purpose
Missing Data Imputation Model in the [[D-Series]]. Layer: **AI/ML**.

## Formula

### LaTeX
$$\hat{x}_t = \sum_{k \in KNN} w_k x_k, \quad w_k = 1/d_k$$

### Description
Missing or invalid sensor readings imputed using K-nearest neighbours (in time and space) with inverse-distance weighting. Falls back to archetype profile if no neighbours available.

### Method
KNN, EM algorithm, forward-fill with decay

## Weights & Parameters

- **W-D06-K:** K=5 nearest neighbours (3 temporal + 2 spatial)
- **W-D06-MAX_GAP:** Max imputation gap: 4h (16 timesteps). Beyond this → alert, use archetype profile
