# D08 — Model Drift Detection & Retraining Trigger

## Purpose
Model Drift Detection & Retraining Trigger in the [[D-Series]]. Layer: **AI/ML**.

## Formula

### LaTeX
$$drift\_flag = CUSUM(e_t) > CUSUM_{thresh}$$

### Description
Detects concept drift in model prediction errors using CUSUM (cumulative sum control chart). When error mean shifts significantly, triggers model retraining.

### Method
CUSUM, ADWIN, prediction error monitoring
