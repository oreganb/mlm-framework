# G03 — Frequency Deviation Model

## Purpose
Frequency Deviation Model in the [[G-Series]]. Layer: **Deterministic**.

## Formula

### LaTeX
$$\Delta f(t) = \frac{1}{2H} \int_0^t (\Delta P_{mech} - \Delta P_{elec}) \, d\tau$$

### Description
Frequency deviation from power imbalance using swing equation. H = system inertia constant. Used for inertia-aware dispatch in islanding mode.

### Method
Swing equation, droop response
