# G04 — Voltage Stability Model

## Purpose
Voltage Stability Model in the [[G-Series]]. Layer: **Deterministic**.

## Formula

### LaTeX
$$V_{pu} = 1 - \frac{R P + X Q}{V_{nom}^2}$$

### Description
Simplified voltage at load bus from active (P) and reactive (Q) power flows. Checks PQ feasibility for proposed dispatch.

### Method
PQ feasibility, static voltage stability

## AI Context (Cursor Alignment)
If PQ_feasible=False, C01 must reduce dispatch. Voltage violations are most common during high export (low load + high PV). G04 check must run before finalising C01 dispatch.
