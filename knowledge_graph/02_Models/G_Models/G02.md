# G02 — Grid Import/Export Limits Model

## Purpose
Grid Import/Export Limits Model in the [[G-Series]]. Layer: **Deterministic**.

## Formula

### LaTeX
$$P_{import} \leq P_{import,max}, \quad P_{export} \leq P_{export,max}$$

### Description
Enforces DSO-mandated import and export limits at the building connection point. Import limit from fuse rating; export limit from G11 hosting capacity.

### Method
Regulatory and technical constraints

## Weights & Parameters

- **W-G02-IMPORT:** Typical Irish residential: 12kW (3×63A fuse at 230V)
- **W-G02-EXPORT:** Export limit: 6kW single-phase (DSO standard)
