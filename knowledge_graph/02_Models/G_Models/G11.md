# G11 — DER Hosting Capacity Model

## Purpose
DER Hosting Capacity Model in the [[G-Series]]. Layer: **Deterministic**.

## Formula

### LaTeX
$$HC_{DER} = \max P_{DER} : V_{bus} \in [V_{min}, V_{max}] \, \forall \, load\_scenarios$$

### Description
Maximum DER power that can be connected to the feeder without violating voltage limits across all load/generation scenarios.

### Method
Incremental hosting capacity analysis
