# G07 — Microgrid Islanding Detection Model

## Purpose
Microgrid Islanding Detection Model in the [[G-Series]]. Layer: **Deterministic**.

## Formula

### LaTeX
$$island\_flag = \mathbf{1}[|df/dt| > ROCOF_{thresh}] \lor \mathbf{1}[|\Delta f| > \Delta f_{thresh}]$$

### Description
Island detection via ROCOF (rate of change of frequency) exceeding 1 Hz/s or frequency deviation exceeding ±0.5 Hz from nominal 50 Hz.

### Method
Frequency/voltage threshold, rate-of-change of frequency (ROCOF)
