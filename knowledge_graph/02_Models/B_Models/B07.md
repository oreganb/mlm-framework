# B07 — Multi-Battery Coordination Model

## Purpose
Multi-Battery Coordination Model in the [[B-Series]]. Layer: **AI/ML**.

## Formula

### LaTeX
$$\min \sum_i C_{deg,i}(P_{bat,i}) \text{ s.t. } \sum_i P_{bat,i} = P_{target}$$

### Description
Coordinates multiple battery units to meet a target aggregate charge/discharge power while minimising total degradation cost across units.

### Method
MILP, priority-based dispatch
