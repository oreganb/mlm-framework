# F02 — Transfer Learning Adapter

## Purpose
Transfer Learning Adapter in the [[F-Series]]. Layer: **AI/ML**.

## Formula

### LaTeX
$$\theta_{adapted} = \arg\min_\theta \mathcal{L}_{target}(\theta) \text{ starting from } \theta_{source}$$

### Description
Adapts pre-trained models from one building/site to a new building with limited local data. Source model trained on similar archetype buildings.

### Method
Fine-tuning, domain adaptation

- **W-F02-LR:** Fine-tune learning rate = 0.0001 (10× smaller than training)
