# F03 — Differential Privacy Layer

## Purpose
Differential Privacy Layer in the [[F-Series]]. Layer: **AI/ML**.

## Formula

### LaTeX
$$\tilde{\theta}_i = \theta_i + \mathcal{N}(0, \sigma_{DP}^2 \mathbf{I})$$

### Description
Adds calibrated Gaussian noise to local model weights before sharing with aggregator (F01). Noise σ_DP calibrated to provide (ε_DP, δ_DP)-differential privacy.

### Method
Gaussian mechanism, DP-SGD
