# F06 — Data Residency Controller

## Purpose
Data Residency Controller in the [[F-Series]]. Layer: **AI/ML**.

## Formula

### LaTeX
$$data\_stays\_local = \mathbf{1}[data\_type \in local\_only\_set]$$

### Description
Enforces data residency policy: raw energy consumption, occupancy, and sensor data never leave the building or island. Only model weights and aggregate statistics are transmitted.

### Method
Data governance rule engine

- **W-F06-ALLOWED:** Allowed to transmit: model weights, aggregated community statistics (≥5 buildings)
