# F01 — Federated Averaging Model

## Purpose
Federated Averaging Model in the [[F-Series]]. Layer: **AI/ML**.

## Formula

### LaTeX
$$\theta_{global}^{(t+1)} = \frac{1}{N} \sum_{i=1}^{N} n_i \theta_i^{(t)} / n$$

### Description
Weighted federated averaging (FedAvg): global model weights computed as data-size-weighted average of local model weights from all participating edge nodes. No raw data leaves edge.

### Method
FedAvg, secure aggregation
