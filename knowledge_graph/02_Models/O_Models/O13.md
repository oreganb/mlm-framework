# O13 — Household Archetype Classification Model

## Purpose
Household Archetype Classification Model in the [[O-Series]]. Layer: **Stochastic**.

## Formula

### LaTeX
$$archetype = \arg\max_k P(x | \mu_k, \Sigma_k)$$

### Description
Classifies each household into one of N_arch=6 archetypes based on building type, occupancy pattern, income, and technology stack. Archetype drives model defaults.

### Method
K-means clustering, supervised classification
