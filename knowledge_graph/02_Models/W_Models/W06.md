# W06 — Humidity & Dew Point Model

## Purpose
Humidity & Dew Point Model in the [[W-Series]]. Layer: **Stochastic**.

## Formula

### LaTeX
$$T_{dew} = \frac{b \cdot \gamma(T, RH)}{a - \gamma(T, RH)}, \quad \gamma = \ln(RH/100) + \frac{aT}{b+T}$$

### Description
Dew point derived from Magnus formula using temperature and relative humidity. a=17.625, b=243.04°C (August-Roche-Magnus constants).

### Method
Magnus formula, regression
