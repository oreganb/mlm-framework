# W12 — UV Index Model

## Purpose
UV Index Model in the [[W-Series]]. Layer: **Stochastic**.

## Formula

### LaTeX
$$UV_{index} = 0.004 \cdot G_{POA}^{0.98}$$

### Description
UV index estimated from plane-of-array irradiance. Used for PV degradation lifetime modelling.
