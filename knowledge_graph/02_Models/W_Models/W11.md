# W11 — Fog & Visibility Model

## Purpose
Fog & Visibility Model in the [[W-Series]]. Layer: **Stochastic**.

## Formula

### LaTeX
$$P(fog) = \sigma(\beta_0 + \beta_1(T_{out} - T_{dew}) + \beta_2 v_{wind})$$

### Description
Fog probability when dew point depression (T-T_dew) < 2°C and wind speed < 3 m/s.

### Method
Logistic regression on dew point depression
