# W04 — Rainfall & Precipitation Model

## Purpose
Rainfall & Precipitation Model in the [[W-Series]]. Layer: **Stochastic**.

## Formula

### LaTeX
$$P(rain=1|X) = \sigma(\beta_0 + \beta_1 P_{atm} + \beta_2 RH + \beta_3 v_{wind})$$

### Description
Rain probability via logistic regression on atmospheric predictors. Intensity modelled as Poisson process when rain flag is true.

### Method
Poisson process, logistic regression

## Weights & Parameters

- **W-W04-B0:** β_0 = -1.2 (Irish baseline rain probability)
- **W-W04-B1:** β_1 = -0.04/hPa
- **W-W04-B2:** β_2 = 0.03/%
