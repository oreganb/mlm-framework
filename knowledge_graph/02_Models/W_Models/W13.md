# W13 — Seasonal Pattern Model

## Purpose
Seasonal Pattern Model in the [[W-Series]]. Layer: **Stochastic**.

## Formula

### LaTeX
$$S(t) = A_0 + \sum_{k=1}^{K} A_k \cos\left(\frac{2\pi k t}{T_{year}}\right) + B_k \sin\left(\frac{2\pi k t}{T_{year}}\right)$$

### Description
Annual seasonal cycle decomposed into K=3 Fourier harmonics. Modulates baseline for all W-Series models.

### Method
Fourier decomposition, calendar features
