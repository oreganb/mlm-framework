# T10 — Thermal Comfort Index Model (PMV/PPD)

## Purpose
Thermal Comfort Index Model (PMV/PPD) in the [[T-Series]]. Layer: **Deterministic**. Part of the building thermal envelope and HVAC modelling stack.

## Formula

### LaTeX
$$PMV = (0.303 e^{-0.036M} + 0.028) \cdot L$$

### Description
Predicted Mean Vote (PMV) from Fanger model. L = thermal load = metabolic rate M minus heat losses. PMV range: -3 (cold) to +3 (hot). Acceptable: -0.5 to +0.5 (ISO 7730 Category A).

### Method
Fanger PMV, ISO 7730
