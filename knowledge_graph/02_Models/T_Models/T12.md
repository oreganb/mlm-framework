# T12 — Multi-Zone Heat Transfer Model

## Purpose
Multi-Zone Heat Transfer Model in the [[T-Series]]. Layer: **Deterministic**. Part of the building thermal envelope and HVAC modelling stack.

## Formula

### LaTeX
$$C_j \frac{dT_j}{dt} = \sum_{i \neq j} \frac{T_i - T_j}{R_{ij}} + \frac{T_{out} - T_j}{R_{env,j}} + Q_{HVAC,j}$$

### Description
Multi-zone extension of T01: N zones modelled as RC network with inter-zone thermal resistances R_ij. Zone coupling captures heat flow between rooms.

### Method
multi-node RC network

## Weights & Parameters

- **W-T12-RIJ:** R_ij (internal wall) ≈ 0.5 K/W (typical partition wall)
- **W-T12-ZONES:** Default: invoke T12 when DS-BUILDING.zone_count > 1
