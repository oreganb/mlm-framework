# T11 — Building Fabric Archetype Lookup Model

## Purpose
Building Fabric Archetype Lookup Model in the [[T-Series]]. Layer: **Deterministic**. Part of the building thermal envelope and HVAC modelling stack.

## Formula

### LaTeX
$$(R_{env}, C_{zone}) = f_{archetype}(type, era, insulation)$$

### Description
Looks up RC thermal parameters for T01 based on building type, construction era, and insulation status. Parameterises the entire T-Series for a building with limited data.

### Method
classification + lookup table

## Weights & Parameters

- **W-T11-R-PRE1980:** R_env = 0.15 K/W pre-1980 uninsulated masonry
- **W-T11-R-POST2010:** R_env = 0.80 K/W post-2010 A-rated
- **W-T11-C-DETACHED:** C_zone = 4.2×10⁶ J/K detached (larger volume)
- **W-T11-C-APT:** C_zone = 1.8×10⁶ J/K apartment (smaller volume)
