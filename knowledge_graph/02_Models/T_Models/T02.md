# T02 — HVAC COP Model

## Purpose
HVAC COP Model in the [[T-Series]]. Layer: **Deterministic**. Part of the building thermal envelope and HVAC modelling stack.

## Formula

### LaTeX
$$COP(T_{in}, T_{out}) = COP_{rated} \cdot [1 + \beta_{COP}(T_{out} - T_{ref})]$$

### Description
COP varies with outdoor temperature. β_COP = -0.08/°C (linear fit). Base COP_rated = 3.2 at T_out = 7°C.

### Method
regression, manufacturer curves
