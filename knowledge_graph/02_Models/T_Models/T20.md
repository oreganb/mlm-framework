# T20 — Surrogate Thermal Model (Fast ANN)

## Purpose
Surrogate Thermal Model (Fast ANN) in the [[T-Series]]. Layer: **AI/ML**. Part of the building thermal envelope and HVAC modelling stack.

## Formula

### LaTeX
$$T_{in}^{surr}(t+1) = f_{ANN}(T_{in}(t), T_{out}(t), G_{POA}(t), Q_{HVAC}(t))$$

### Description
Trained ANN replacing T01 RC model for fast inference. 3-layer MLP trained on T01 outputs. Inference 100× faster than T01 on Jetson Nano — used for real-time MPC inner loop.

### Method
ANN surrogate for T01, neural network
