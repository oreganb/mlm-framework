# L10 — Load Forecasting Model

## Purpose
Load Forecasting Model in the [[L-Series]]. Layer: **Deterministic**.

## Formula

### LaTeX
$$\hat{P}(t+h) = f_{LSTM}(P_{hist}, T_{out,forecast}, occ_{forecast})$$

### Description
Short-term load forecast for horizon h (1h, 3h, 24h) using LSTM trained on historical load, weather forecast, and occupancy forecast.

### Method
LSTM, gradient boosting
