# L05 — Lighting Load Model

## Purpose
Lighting Load Model in the [[L-Series]]. Layer: **Deterministic**.

## Formula

### LaTeX
$$P_{lighting}(t) = P_{rated,light} \cdot occ\_fraction(t) \cdot (1 - daylight\_factor(t))$$

### Description
Lighting load from occupancy fraction and daylight suppression factor.

### Method
Occupancy and daylight model
