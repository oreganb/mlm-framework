# L11 — Peak Load Detection & Forecasting Model

## Purpose
Peak Load Detection & Forecasting Model in the [[L-Series]]. Layer: **Deterministic**.

## Formula

### LaTeX
$$P_{peak} = \max_{t \in day} P_{net}(t), \quad peak\_flag = \mathbf{1}[P_{peak} > P_{thresh}]$$

### Description
Daily peak load detection and forecasting. Triggers pre-emptive storage charging before anticipated peak periods.

### Method
Threshold detection, extreme value analysis
