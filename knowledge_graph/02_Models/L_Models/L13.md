# L13 — Demand Response Adjusted Load Model

## Purpose
Demand Response Adjusted Load Model in the [[L-Series]]. Layer: **Deterministic**.

## Formula

### LaTeX
$$P_{DR}(t) = P_{total}(t) - P_{shed}(t) - P_{shifted}(t) + P_{shift\_restore}(t)$$

### Description
DR-adjusted load accounting for shed loads (removed), shifted loads (moved earlier/later), and restoration of shifted loads at their rescheduled time.

### Method
Event-based load adjustment
