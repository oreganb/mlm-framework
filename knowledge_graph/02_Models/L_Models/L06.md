# L06 — DHW Load Model

## Purpose
DHW Load Model in the [[L-Series]]. Layer: **Deterministic**.

## Formula

### LaTeX
$$P_{DHW,elec}(t) = (Q_{DHW}(t) + Q_{cyl,loss}(t)) / \eta_{heater}$$

### Description
Electrical power for DHW heating: draw demand plus cylinder standing losses, divided by heater efficiency.

### Method
From O04 DHW demand, T08 cylinder losses
