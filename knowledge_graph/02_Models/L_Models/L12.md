# L12 — Load Profile Clustering Model

## Purpose
Load Profile Clustering Model in the [[L-Series]]. Layer: **Deterministic**.

## Formula

### LaTeX
$$archetype\_profile = \arg\min_k DTW(P_{hist}, centroid_k)$$

### Description
Clusters building load profiles using Dynamic Time Warping distance to K=8 load archetypes. Assigns each building its closest archetype for initialising L08 base load.

### Method
K-means, DTW distance
