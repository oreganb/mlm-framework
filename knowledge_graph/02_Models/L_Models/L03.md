# L03 — HVAC Load Decomposition Model

## Purpose
HVAC Load Decomposition Model in the [[L-Series]]. Layer: **Deterministic**.

## Formula

### LaTeX
$$P_{HVAC}(t) = Q_{HVAC}(t) / COP(T_{in}, T_{out})$$

### Description
Electrical power draw of HVAC system from thermal output and COP. Derived from T01 and T02 rather than measured directly.

### Method
Physics-based from T-Series
