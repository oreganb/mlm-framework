# L08 — Base Load Model (Non-Controllable)

## Purpose
Base Load Model (Non-Controllable) in the [[L-Series]]. Layer: **Deterministic**.

## Formula

### LaTeX
$$P_{base}(t) = P_{base,archetype}(t) \cdot income\_factor \cdot event\_factor$$

### Description
Non-controllable base load: always-on appliances, standby power, unshiftable consumption. Modulated by income (O14) and events (O12).

### Method
Regression, archetype profiles
