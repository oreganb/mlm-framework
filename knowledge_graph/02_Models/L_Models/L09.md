# L09 — Flexible Load Identification Model

## Purpose
Flexible Load Identification Model in the [[L-Series]]. Layer: **Deterministic**.

## Formula

### LaTeX
$$flex\_flag_i = \mathbf{1}[P_i \in shiftable\_set \land smart\_enabled_i]$$

### Description
Identifies which loads are shiftable (have time flexibility) and smart-enabled. Produces boolean flags per load used by C01 for the P_shift variable.

### Method
Classification, rule-based flagging
