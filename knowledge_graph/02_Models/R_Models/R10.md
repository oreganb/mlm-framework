# R10 — Self-Consumption Optimisation Model

## Purpose
Self-Consumption Optimisation Model in the [[R-Series]]. Layer: **Deterministic**.

## Formula

### LaTeX
$$\max_{P_{bat}} \int_T P_{PV}(t) \cdot \mathbf{1}[consumed\_locally] \, dt$$

### Description
Maximises fraction of PV generation consumed locally (not exported) by optimal battery charging. A simplified LP version of C01 focused only on self-consumption.

### Method
LP dispatch
