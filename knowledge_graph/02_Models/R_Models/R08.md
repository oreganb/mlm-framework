# R08 — Hybrid PV + Battery Combined Output Model

## Purpose
Hybrid PV + Battery Combined Output Model in the [[R-Series]]. Layer: **Deterministic**.

## Formula

### LaTeX
$$P_{net\_DER}(t) = P_{PV}(t) + P_{wind}(t) - P_{bat,chg}(t) + P_{bat,dis}(t)$$

### Description
Net DER output at the building PCC: PV plus wind minus battery charging plus battery discharging. This is what the meter sees.

### Method
Power flow summation
