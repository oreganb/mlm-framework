# R04 — MPPT (Maximum Power Point Tracking) Model

## Purpose
MPPT (Maximum Power Point Tracking) Model in the [[R-Series]]. Layer: **Deterministic**.

## Formula

### LaTeX
$$P_{MPPT} = V_{mpp} \cdot I_{mpp} = P_{STC} \cdot f(G_{POA}, T_{cell})$$

### Description
MPPT surrogate model tracking the maximum power point. Efficiency penalty applied when inverter MPPT fails to track due to partial shading.

### Method
Perturb & Observe algorithm surrogate, regression

## Weights & Parameters

- **W-R04-ETA_MPPT:** η_MPPT = 0.98 (modern inverter tracking efficiency)
- **W-R04-SHADE:** Partial shading penalty: up to 30% power loss per shaded string
