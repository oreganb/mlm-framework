# R06 — PV Curtailment Model

## Purpose
PV Curtailment Model in the [[R-Series]]. Layer: **Deterministic**.

## Formula

### LaTeX
$$P_{PV,curtailed}(t) = \max(0, P_{MPPT}(t) - P_{export\_limit}(t))$$

### Description
PV output curtailed when grid export limit is hit (DSO constraint) or when battery/load cannot absorb surplus. Curtailed energy is wasted.

### Method
Grid limit rule, clipping
