# R09 — DER Availability Stochastic Model

## Purpose
DER Availability Stochastic Model in the [[R-Series]]. Layer: **Deterministic**.

## Formula

### LaTeX
$$avail_i(t) \sim Bernoulli(p_{avail,i}), \quad p_{avail} \sim Beta(\alpha, \beta)$$

### Description
Stochastic availability of each DER (PV inverter, wind turbine, battery) modelled as Bernoulli draws from Beta-distributed availability probability.

### Method
Bernoulli/Beta availability, Monte Carlo
