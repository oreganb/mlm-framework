# C10 — Governance & Safety Layer

## Purpose
Governance & Safety Layer in the [[C-Series]]. Layer: **RL/Q-Learning**.

## Formula

### LaTeX
$$u_{governed} = \begin{cases} u_{proposed} & \text{if } RD \geq RD_{min} \land safety\_ok \\ \alpha_{derate} \cdot u_{proposed} & \text{if } RD < RD_{min} \\ 0 & \text{if } storm\_flag \lor override\_flag \end{cases}$$

### Description
Three-state governance: allow (full dispatch), derate (scale down by α), or block (zero dispatch). Applied to every C-Series output before execution.

### Method
Rule-based gating, constraint enforcement

## Weights & Parameters

- **W-C10-RDMIN:** RD_min = 0.1 (below this, asset provides negligible flex — derate)
- **W-C10-ALPHA:** α_derate = 0.5 (50% derating when RD borderline)
- **W-C10-RATELIMIT:** Rate limit: max ±5% change per timestep (prevents oscillation)
