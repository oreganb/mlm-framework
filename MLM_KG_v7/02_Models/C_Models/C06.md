# C06 — Actor-Critic HVAC Controller

## Purpose
Actor-Critic HVAC Controller in the [[C-Series]]. Layer: **RL/Q-Learning**.

## Formula

### LaTeX
$$L_{CLIP}(\theta) = E[\min(r_t A_t, \text{clip}(r_t, 1-\epsilon, 1+\epsilon) A_t)]$$

### Description
PPO actor-critic for continuous HVAC setpoint control. Actor outputs setpoint T_setpoint; critic estimates value function. Clipped surrogate objective prevents large policy updates.

### Method
Proximal Policy Optimisation (PPO), Actor-Critic
