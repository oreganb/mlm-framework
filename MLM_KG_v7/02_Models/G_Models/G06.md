# G06 — Network Loss Model

## Purpose
Network Loss Model in the [[G-Series]]. Layer: **Deterministic**.

## Formula

### LaTeX
$$P_{loss}(t) = I^2(t) \cdot R_{line} = \frac{P^2_{net}(t)}{V^2_{nom}} \cdot R_{line}$$

### Description
Feeder cable losses from current squared × resistance. Reduces with lower net current (local generation offsets import).

### Method
I²R loss calculation
