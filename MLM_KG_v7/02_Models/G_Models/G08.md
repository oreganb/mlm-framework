# G08 — Fragmented Islanding Segment Model

## Purpose
Fragmented Islanding Segment Model in the [[G-Series]]. Layer: **Deterministic**.

## Formula

### LaTeX
$$segment_i = \{buildings : \sum P_{gen,i} \approx \sum P_{load,i}\}$$

### Description
Identifies which buildings can form a viable self-sustaining island segment based on local generation-load balance. Enables fragmented (partial) islanding.

### Method
Graph topology, load-generation matching
