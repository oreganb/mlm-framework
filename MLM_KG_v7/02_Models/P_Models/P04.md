# P04 — Self-Consumption KPI Model

## Purpose
Self-Consumption KPI Model in the [[P-Series]]. Layer: **RL/Q-Learning**.

## Formula

### LaTeX
$$SC\_ratio = \frac{E_{gen,consumed\_locally}}{E_{gen,total}} = 1 - \frac{E_{export}}{E_{PV} + E_{wind}}$$

### Description
Self-consumption ratio: fraction of local generation consumed on-site rather than exported. Target >80% per SEAI guidance.

### Method
Energy accounting ratio
