# T19 — Building EPC Classification Model

## Purpose
Building EPC Classification Model in the [[T-Series]]. Layer: **Deterministic**. Part of the building thermal envelope and HVAC modelling stack.

## Formula

### LaTeX
$$EPC\_score = 100 - \sum_i w_i \cdot U_i \cdot A_i - penalties_{HVAC}$$

### Description
EPC score from fabric U-values and HVAC system efficiency. Each building element (wall, roof, floor, window) weighted by area and U-value contribution to heat loss.

### Method
U-value regression, EPC scoring
