# T15 — Gas/Oil Boiler Thermal Model

## Purpose
Gas/Oil Boiler Thermal Model in the [[T-Series]]. Layer: **Deterministic**. Part of the building thermal envelope and HVAC modelling stack.

## Formula

### LaTeX
$$Q_{boiler} = P_{gas} \cdot \eta_{boiler}, \quad P_{gas} = \frac{Q_{demand}}{\eta_{boiler}}$$

### Description
Boiler thermal output from fuel input and combustion efficiency. η_boiler = 0.90 for condensing, 0.78 for non-condensing. 40% of ODEON buildings have gas/oil boilers.

### Method
combustion efficiency, RC
