# M08 — Carbon Intensity Forecast Model

## Purpose
Carbon Intensity Forecast Model in the [[M-Series]]. Layer: **Stochastic**.

## Formula

### LaTeX
$$EF(t) = EF_{base} + \delta_{RES}(t) \cdot \Delta EF_{RES} + \epsilon_t$$

### Description
Marginal carbon emission factor forecast. Decreases when renewable generation is high (high RES → lower marginal EF). Base EF = 0.295 kgCO₂/kWh (Ireland 2024).

### Method
ARIMA on EirGrid carbon intensity signal, regression
