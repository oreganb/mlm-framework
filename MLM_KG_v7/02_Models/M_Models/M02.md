# M02 — Intraday Price Model

## Purpose
Intraday Price Model in the [[M-Series]]. Layer: **Stochastic**.

## Formula

### LaTeX
$$P^{ID}(t) = P^{DA}(t) + \Delta_{ID}(t), \quad \Delta_{ID} \sim \mathcal{N}(0, \sigma^2_{ID})$$

### Description
Intraday price as day-ahead plus zero-mean correction term. σ_ID captures intraday volatility driven by real-time supply-demand imbalance.

### Method
ARIMA, real-time price signal processing
