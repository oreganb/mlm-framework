# F04 — Cross-Site Calibration Model

## Purpose
Cross-Site Calibration Model in the [[F-Series]]. Layer: **AI/ML**.

## Formula

### LaTeX
$$x_{calibrated,j} = x_{raw,j} + b_j, \quad b_j = \bar{x}_{reference} - \bar{x}_{j}$$

### Description
Calibrates sensor readings across buildings and sites to a common reference to remove systematic sensor-to-sensor bias. Enables cross-building model transfer.

### Method
Bias correction, domain adaptation
