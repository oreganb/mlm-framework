# F05 — Model Versioning & Rollback Controller

## Purpose
Model Versioning & Rollback Controller in the [[F-Series]]. Layer: **AI/ML**.

## Formula

### LaTeX
$$active\_version = v^* = \arg\max_v Performance(v, \mathcal{D}_{validation})$$

### Description
Maintains version history of all ML models. Automatically rolls back to previous version if new federated round degrades performance on held-out validation set.

### Method
Version control, performance monitoring
