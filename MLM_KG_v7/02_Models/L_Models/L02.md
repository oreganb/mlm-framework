# L02 — NILM Disaggregation Model

## Purpose
NILM Disaggregation Model in the [[L-Series]]. Layer: **Deterministic**.

## Formula

### LaTeX
$$P_{total}(t) = \sum_{i=1}^{N} P_i(t) + \epsilon_{NILM}(t)$$

### Description
Disaggregates aggregate smart meter signal into per-appliance contributions using autoencoder + HMM. ε_NILM is residual disaggregation error.

### Method
Non-Intrusive Load Monitoring, autoencoder, HMM
