# L01 — Net Load Model

## Purpose
Net Load Model in the [[L-Series]]. Layer: **Deterministic**.

## Formula

### LaTeX
$$P_{net}(t) = P_{total}(t) - P_{PV}(t) - P_{wind}(t)$$

### Description
Net load at the building point of connection: total consumption minus local renewable generation. Positive = import, negative = export.

### Method
Kirchhoff power balance
