# O01 — Occupancy Probability Model

## Purpose
Occupancy Probability Model in the [[O-Series]]. Layer: **Stochastic**.

## Formula

### LaTeX
$$P_{occ}(t) = \sum_{j \in S} \pi_j(t) \cdot \mathbf{1}[S_j \neq unoccupied]$$

### Description
HMM with states {Unoccupied=0, Partial=1, Full=2}. Transition probabilities π_j learned from PIR, BLE, and WiFi sensor data fusion.

### Method
Hidden Markov Model (HMM), PIR/BLE/WiFi fusion
