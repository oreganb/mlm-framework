# O15 — Occupant Override & HMI Signal Model

## Purpose
Occupant Override & HMI Signal Model in the [[O-Series]]. Layer: **Stochastic**.

## Formula

### LaTeX
$$u_{final}(t) = override\_signal(t) \oplus u_{auto}(t)$$

### Description
Merges automated dispatch signals from C01 with manual occupant overrides from HMI (thermostat, app). Override takes priority for duration D_override.

### Method
Rule-based, HMI event handler
