# O12 — Holiday & Event Pattern Model

## Purpose
Holiday & Event Pattern Model in the [[O-Series]]. Layer: **Stochastic**.

## Formula

### LaTeX
$$load\_factor(t) = 1 + \delta_{event}(t) \cdot \Delta_{load,event}$$

### Description
Load and occupancy modulation factors for public holidays, school holidays, sports events, and seasonal gatherings. Irish and local Aran Islands calendar.

### Method
Calendar lookup, event classification

## Weights & Parameters

- **W-O12-XMAS:** Christmas day load factor = 1.7 (highest annual peak)
- **W-O12-SUMMER:** Extended summer holiday (3+ weeks): load factor 0.6
