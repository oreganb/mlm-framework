# O03 — Thermal Comfort Feedback Model

## Purpose
Thermal Comfort Feedback Model in the [[O-Series]]. Layer: **Stochastic**.

## Formula

### LaTeX
$$override = \mathbf{1}[PMV > PMV_{thresh}] \lor \mathbf{1}[T_{in} < T_{min,occ}]$$

### Description
Occupant override triggered when PMV exceeds comfort threshold or indoor temperature falls below occupied minimum. Override signal sent to C10 governance layer.

### Method
PMV-based threshold, user override signal

## Weights & Parameters

- **W-O03-PMVTHRESH:** PMV override threshold: |PMV| > 1.0 (outside comfortable range)
- **W-O03-TMIN:** Occupied T_min = 19°C (stricter than unoccupied 16°C)
