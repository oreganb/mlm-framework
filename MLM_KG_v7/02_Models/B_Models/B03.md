# B03 — EV Arrival/Departure Markov Model

## Purpose
EV Arrival/Departure Markov Model in the [[B-Series]]. Layer: **AI/ML**.

## Formula

### LaTeX
$$P(EV\_state_{t+1} | EV\_state_t, schedule) = \Pi_{schedule}$$

### Description
Markov chain for EV present/absent state transitions, conditioned on occupant schedule type. Determines when EV is available for charging and V2G.

### Method
Markov chain, schedule-conditioned
