# B09 — Federated Battery Learning Model

## Purpose
Federated Battery Learning Model in the [[B-Series]]. Layer: **AI/ML**.

## Formula

### LaTeX
$$\theta_{LSTM,i} \leftarrow \theta_{global} + \Delta\theta_{local,i}$$

### Description
Each building's B01 LSTM model fine-tuned from global weights θ_global (from F01) with local data. Personalises to battery chemistry and usage patterns.

### Method
Federated averaging, transfer learning
