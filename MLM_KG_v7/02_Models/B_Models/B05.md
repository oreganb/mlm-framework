# B05 — V2G Availability Model

## Purpose
V2G Availability Model in the [[B-Series]]. Layer: **AI/ML**.

## Formula

### LaTeX
$$V2G\_avail = EV\_present \land (SoC > SoC_{V2G,min}) \land grid\_allows\_V2G$$

### Description
V2G (Vehicle-to-Grid) availability as conjunction of: EV physically present, SoC above V2G minimum (preserving departure range), and grid/contract permission.

### Method
Multi-constraint Boolean gate
