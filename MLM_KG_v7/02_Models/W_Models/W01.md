# W01 — Solar Irradiance Model

## Purpose
Estimates the solar irradiance at the plane of array (G_POA) for a given site, combining astronomical geometry, atmospheric correction, cloud cover scaling, and ensemble uncertainty quantification. Primary input to [[R01 PV Generation Model]] and [[T01 Thermal RC Model]] (solar gain term).

## Layer
**Stochastic** — outputs a probability distribution over G_POA, not a point estimate. Quantiles (p10/p50/p90) are passed to the determinization bridge.

---

## Formula

### LaTeX
$$G_{POA} = G_{DNI} \cos(\theta_z) + G_{DHI} + G_{reflected}$$

**With cloud attenuation:**
$$G_{POA,cloud} = G_{POA,clear} \times (1 - k_{cloud} \times f_{OKTAS})$$

**Ensemble version:**
$$G_{POA}^{(\omega)} = G_{POA,cloud} + \epsilon^{(\omega)}, \quad \epsilon \sim \mathcal{N}(0, \sigma_{irr}^2)$$

### Python Pseudocode
```python
def compute_G_POA(G_DNI, G_DHI, theta_z, OKTAS, sigma_irr, n_ensemble=50):
    """
    Inputs:
      G_DNI    : Direct Normal Irradiance (W/m²) from DS-NWP or DS-LOCAL-SENSOR
      G_DHI    : Diffuse Horizontal Irradiance (W/m²) from DS-NWP
      theta_z  : Solar zenith angle (degrees) from W08
      OKTAS    : Cloud cover score (0-8) from W05
      sigma_irr: Forecast uncertainty (W/m²), from DS-NWP model spread
      n_ensemble: Number of ensemble members (default 50)
    """
    cos_z = cos(radians(theta_z))
    G_POA_clear = G_DNI * cos_z + G_DHI  # [w=1.0] direct + [w=1.0] diffuse
    
    k_cloud = 0.11   # [w=0.11, calibrated: Perez model, see W-K-CLOUD]
    f_OKTAS = OKTAS / 8.0   # normalise 0-8 to 0-1
    
    G_POA_cloud = G_POA_clear * (1.0 - k_cloud * f_OKTAS)
    
    # Ensemble perturbation
    eps = normal(0, sigma_irr, n_ensemble)
    G_POA_ensemble = G_POA_cloud + eps  # shape: (n_ensemble,)
    
    return {
        "p10": percentile(G_POA_ensemble, 10),
        "p50": percentile(G_POA_ensemble, 50),
        "p90": percentile(G_POA_ensemble, 90),
        "ensemble": G_POA_ensemble
    }
```

### Plain English
Direct sunlight hitting the panel (G_DNI, scaled by the cosine of the sun's angle) plus diffuse sky radiation (G_DHI) gives the clear-sky irradiance. This is then reduced based on cloud cover (the OKTAS score, scaled from 0–8 to 0–1, multiplied by the cloud attenuation coefficient k_cloud = 0.11). Fifty random perturbations are added to represent forecast uncertainty, yielding a full distribution.

## AI Context
When implementing W01:
- `G_DNI` and `G_DHI` must be fetched from `DS-NWP.irradiance_dni` and `DS-NWP.irradiance_dhi` respectively — never estimated or computed internally
- Always call [[W08]] first to get `theta_z` before computing `cos_z`
- Always call [[W05]] first to get `OKTAS` before applying cloud attenuation
- Return all four outputs (p10, p50, p90, ensemble) as a named dict/struct
- `k_cloud = 0.11` is not tunable without re-calibration — see [[W-W01-KCLOUD]] in [[06_Weighting/Weight Registry]]
