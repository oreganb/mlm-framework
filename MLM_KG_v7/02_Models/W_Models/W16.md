# W16 — Day-Ahead Forecast Model (T+24h)

## Purpose
Day-Ahead Forecast Model (T+24h) in the [[W-Series]]. Layer: **Stochastic**.

## Formula

### LaTeX
$$\hat{G}_{DA}(t+24h) = f_{GBM}(NWP_{T+24}, S_{seasonal}, T_{lag1})$$

### Description
Day-ahead irradiance forecast using GBM post-processing of NWP output with seasonal features and lagged observations as predictors.

### Method
NWP post-processing, gradient boosting, ARIMA
