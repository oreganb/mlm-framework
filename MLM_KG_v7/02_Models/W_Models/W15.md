# W15 — Nowcasting Model (T+0 to T+60min)

## Purpose
Nowcasting Model (T+0 to T+60min) in the [[W-Series]]. Layer: **Stochastic**.

## Formula

### LaTeX
$$\hat{G}_{POA}(t+\tau) = f_{LSTM}(G_{hist}, radar_{t})$$

### Description
Sub-hourly irradiance nowcast using radar cloud motion extrapolation fused with LSTM on historical irradiance sequence. Horizon τ = 0–60 minutes.

### Method
Radar extrapolation, LSTM
