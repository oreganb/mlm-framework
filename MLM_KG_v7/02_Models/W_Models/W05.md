# W05 — Cloud Cover Model (OKTAS)

## Purpose
Cloud Cover Model (OKTAS) in the [[W-Series]]. Layer: **Stochastic**.

## Formula

### LaTeX
$$OKTAS \in \{0,1,2,3,4,5,6,7,8\}, \quad f_{cloud} = OKTAS/8$$

### Description
Cloud cover classified as OKTAS score 0–8 using gradient boosting on NWP inputs. f_cloud (0–1) is the normalised attenuation factor used in W01.

### Method
ML classification, ARIMA

## Weights & Parameters

- **W-W05-OKTAS-NORM:** f_cloud = OKTAS/8, linear normalisation
- **W-W05-GBM:** GBM with 100 trees, max_depth=4, calibrated on Met Éireann OKTAS archive
