# W10 — Extreme Weather Event Model

## Purpose
Extreme Weather Event Model in the [[W-Series]]. Layer: **Stochastic**.

## Formula

### LaTeX
$$P(extreme) = 1 - e^{-(v_{wind}/v_{thresh})^\alpha}$$

### Description
Probability of extreme wind event using Gumbel distribution tail. Storm flag triggered when wind >18m/s or gusts >25m/s.

### Method
Threshold classification, extreme value theory

## Weights & Parameters

- **W-W10-VTHRESH:** Storm threshold: 18 m/s sustained, 25 m/s gust
- **W-W10-ALPHA:** Gumbel α = 2.1 (fitted, Atlantic storms)
