# D04 — Sensor Fault Detection Model

## Purpose
Sensor Fault Detection Model in the [[D-Series]]. Layer: **AI/ML**.

## Formula

### LaTeX
$$fault\_score_i = IsolationForest(x_i, x_{neighbours})$$

### Description
Multivariate sensor fault detection using Isolation Forest on sensor readings and their spatial/temporal neighbours. Fault score > threshold triggers alert.

### Method
Isolation Forest, LSTM autoencoder
