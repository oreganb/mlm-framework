# D01 — Weather Data QA Gate

## Purpose
Weather Data QA Gate in the [[D-Series]]. Layer: **AI/ML**.

## Formula

### LaTeX
$$quality\_flag = \mathbf{1}[x \in [x_{min}, x_{max}]] \land \mathbf{1}[|\Delta x / \Delta t| < ROC_{max}]$$

### Description
Quality gate on weather sensor inputs: range check (physical limits) and rate-of-change check (implausible sudden changes). Flags suspect and failed readings.

### Method
Range checks, outlier detection, gap flagging
